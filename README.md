## üåü Overview
This repository contains the official source code for the website of **Integration 2026**, the Annual Socio-Cultural and Sports Fest of the **Indian Statistical Institute (ISI), Kolkata**.

Integration is ISI Kolkata's flagship cultural and intellectual festival, uniting thousands of students from across India in a vibrant arena to showcase their talents. The event takes place between **January and March 2026**.

ISI Kolkata itself is an Institute of National Importance, established in 1931 by Professor Prasanta Chandra Mahalanobis, and is globally recognized as a premier center for quantitative sciences.

**Visit the Live Site:**
> [isik-integration-2026.vercel.app](https://www.integrationfest.in/)

---

## ‚ú® Key Features & Event Chapters
The website serves as the central hub for the festival, providing detailed information on all competitive chapters and events:

* **Home:** Dynamic landing page, theme showcase, and event countdown.
* **Events:** Detailed information and registration links for various events, categorized into major chapters:
    * **Cultural Chapter:** Events include **Alaap** (singing event), **Thru Da Lens** (photography contest), **Model United Nations** (MUN), and **Between the Lines** (creative writing).
    * **Sports Chapter:** Tournaments for **Football**, **Cricket**, **Chess**, **Table Tennis**, and **Bridge**.
    * **Tech/Academic Chapter:**
        * **MTRP** (Mathematics Talent Reward Programme).
        * **SPARQ** (a **data analytics-based competition** focusing on data science and statistical reasoning).
    * **Quizzing:** The **Integration Bee (IntBee)**, a multi-round calculus championship.
* **Gallery:** A showcase of photos and videos from previous years.
* **Team & Contact:** Introduction to the organizing committee members and official contact information.

## üöÄ Getting Started

### Prerequisites
You need a modern web browser and a code editor. **Git** and **Node.js** are recommended for contribution.

### Installation

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/Bubu-Droid/isik-integration-2026.git](https://github.com/Bubu-Droid/isik-integration-2026.git)
    cd isik-integration-2026
    ```

2.  **Install dependencies (if applicable):**
    *If you use a package manager (like npm/yarn) for development tools or libraries:*
    ```bash
    # If a package.json file exists:
    npm install 
    # OR
    yarn install
    ```

### Running Locally

1.  Open the `index.html` file in your preferred web browser.
    *Alternatively, use a local server (like VS Code's Live Server extension or `http-server`):*
    ```bash
    # If using http-server (install with npm install -g http-server)
    http-server
    ```
    The site will be accessible at `http://localhost:8080` (or similar).
